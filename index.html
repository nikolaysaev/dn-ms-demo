About this demo

This is a fake storefront built only to demonstrate the CartAssist chatbot. Products, prices and the cart are sample data. The chat widget auto-loads on page load using the hardcoded public key and talks to the dashboard /v1 API; the panel below is a direct-to-core debugging fallback. The full product export lives on the server for backend import and chatbot search.
